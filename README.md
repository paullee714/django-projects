# django-projects
django와 관련된 기능들을 모아봅니다

## 공통 실행 요소
```bash
$ virtualenv venv
$ source venv/bin/actiavte 
(venv) $ pip install django pymysql mysqlclient
# 위의 과정이 모두 끝난 후에 프로젝트 폴더에 들어가 README.md를 참조하여 실행
```
